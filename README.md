# 🛒 LaptopStore Website

## 📌 Giới thiệu

Đây là dự án website bán laptop với đầy đủ chức năng cơ bản:

* Xem danh sách sản phẩm
* Xem chi tiết sản phẩm
* Quản lý sản phẩm (Admin)
* Hệ thống frontend + backend tách biệt

---

## 🛠️ Công nghệ sử dụng

### Frontend

* ReactJS
* HTML, CSS, Bootstrap

### Backend

* (Bạn sửa lại cho đúng: NodeJS / .NET / Java)

### Database

* (MySQL / SQL Server / MongoDB)

### DevOps

* Docker
* Docker Compose

---

## ⚙️ Cách chạy project

### ✅ Cách 1: Chạy bằng Docker (Khuyến khích)

```bash
docker-compose up --build
```

Sau đó truy cập:

* Frontend: http://localhost:3000
* Backend: http://localhost:5000

---

### ✅ Cách 2: Chạy thủ công

#### 🔹 Backend

```bash
cd backend
npm install
npm start
```

#### 🔹 Frontend

```bash
cd frontend
npm install
npm start
```
